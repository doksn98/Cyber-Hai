# Cyber-Hai - 2D Arcade Game :)

Ein neon-grüner Cyber-Hai kämpft in diesem browserbasierten 2D-Arcade-Spiel gegen die Verschmutzung der Meere, indem er Fische frisst und "gehacktem" Müll ausweicht. Das Spiel wurde mit HTML5 Canvas, CSS3 und reinem JavaScript entwickelt und besticht durch ein futuristisches Cyber-Design inklusive Glow-Effekten und einer reaktiven Tastatursteuerung.

## Features
* **Steuerung:** Bewege den Hai mit den Pfeiltasten oder WASD in alle vier Richtungen (oben, unten, links, rechts).
* **Mechanik:** Sammle Punkte durch das Fressen von Fischen, aber verliere eines deiner 3 Leben, wenn du Reifen, Dosen oder Chipstüten berührst.
* **Pause-Funktion:** Nutze die Leertaste, um das Spiel jederzeit zu pausieren.
* **Design:** Custom Assets für den Hai, die Gegner und den Hintergrund sorgen für eine stimmige Atmosphäre.

## Technologien
* HTML5 (Canvas API)
* CSS3 (Flexbox, Animationen)
* JavaScript (ES6+)
